# AHKv2 & LLMs
LLM Performance in AHK v2 Coding Tasks

## Visit the Site
**[https://truecrimedev.github.io/AHKv2_LLMs](https://truecrimedev.github.io/AHKv2_LLMs)**

This project explores using AI assistants like GitHub Copilot, ChatGPT, and Claude for AutoHotkey v2 development.

## Features

### Blog Posts
In-depth articles covering AHK v2.1 alpha releases, OOP patterns, GUI development, prompt engineering, and LLM workflows.

### AHK v2 Wiki
A comprehensive reference for AutoHotkey v2.1 features -- structs, typed arrays, modules, callbacks, and Win32 interop. Written as a single-page knowledge base.

### Code Examples
Runnable `.ahk` scripts in `examples/` demonstrating every feature introduced in alpha releases 22 through 26, plus `Opus48_Example.ahk` -- an idiomatic clipboard-history manager generated by Claude Opus 4.8 (see the accompanying "Claude Opus 4.8: AHK v2 Output Examples" post).

### Live Repos Feed
The sidebar shows the latest AutoHotkey v2 repositories from GitHub, sorted by recent activity.

## Preview Locally

```
python3 -m http.server 8080
```

Open `http://localhost:8080` in a browser. The repos feed requires internet access and works best on GitHub Pages.
