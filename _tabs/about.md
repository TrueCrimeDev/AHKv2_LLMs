# About This Site

This site provides a practical guide to leveraging AI assistants for AutoHotkey v2 development. 

## Purpose

The purpose of this guide is to help developers understand how to effectively use Large Language Models (LLMs) and AI coding agents like GitHub Copilot, ChatGPT, and Claude to accelerate their AutoHotkey v2 development workflow.

## Topics Covered

- AHK v2 syntax differences from v1
- Effective prompting strategies for AI assistants
- Common automation use cases
- Debugging with AI assistance
- Best practices for AI-assisted development
