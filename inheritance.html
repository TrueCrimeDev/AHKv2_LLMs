AHK v2 Object Inheritance

AutoHotkey v2 is prototype-based: every value descends from Any, and inheritance flows through two parallel .Base chains — one for instances (the prototype chain) and one for classes (for static members). Explore the built-in type hierarchy below, then see how a user-defined extends maps onto both chains.
